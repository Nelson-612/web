About

I’m a Master of Science in Computer Science student at Northeastern University with a background in animal science and veterinary care. I enjoy building backend systems, REST APIs, and data-driven applications that are clean, reliable, and easy to maintain.

My transition into computer science was driven by seeing how software and data improve accuracy, efficiency, and decision-making in real-world environments. I’m especially interested in backend development, systems engineering, and scalable applications.

Selected Projects

Animal Species Classification (Deep Learning)

Built a multi-class image classification system to predict animal species from images using transfer learning with a pre-trained VGG-16 convolutional neural network. The model was trained on ~26,000 images across 10 classes and evaluated using accuracy, precision/recall, and a confusion matrix.

Achieved 74% test accuracy and performed class-level error analysis to identify systematic misclassification among visually similar animals such as cow, sheep, and horse.

Tech: Python, TensorFlow, Keras, scikit-learn, Matplotlib

FastAPI · Pydantic · REST API · HTML/CSS/JS · Explainable Rules

A full-stack web application that estimates pet health risks (obesity, dental, and joint) based on user inputs. The backend exposes a FastAPI API with validated Pydantic schemas, and the frontend visualizes scores with clear reasons and recommended care actions.
